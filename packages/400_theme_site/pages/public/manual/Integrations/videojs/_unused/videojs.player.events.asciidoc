== Player

=== Events

[[Eventsended]]
[[endedHeader]]
==== ended

[[endedDescription]]
Fired when video playback ends


[[Eventserror]]
[[errorHeader]]
==== error

[[errorDescription]]
Fired when an error occurs


[[Eventsloadeddata]]
[[loadeddataHeader]]
==== loadeddata

[[loadeddataDescription]]
Fired when the player has downloaded data at the current playback
position


[[Eventsloadedmetadata]]
[[loadedmetadataHeader]]
==== loadedmetadata

[[loadedmetadataDescription]]
Fired when the player has initial duration and dimension information


[[Eventstimeupdate]]
[[timeupdateHeader]]
==== timeupdate

[[timeupdateDescription]]
Fired when the current playback position has changed * During playback
this is fired every 15-250 milliseconds, depending on the playback
technology in use.


[[Eventsuseractive]]
[[useractiveHeader]]
==== useractive

[[useractiveDescription]]
Fired when the user is active, e.g. moves the mouse over the player


[[Eventsuserinactive]]
[[userinactiveHeader]]
==== userinactive

[[userinactiveDescription]]
Fired when the user is inactive, e.g. a short delay after the last mouse
move or control interaction


[[Eventsvolumechange]]
[[volumechangeHeader]]
==== volumechange

[[volumechangeDescription]]
Fired when the volume changes
