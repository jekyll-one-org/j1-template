This document manages the chat interface (prompt inputs) for the AI Agent
Claude SkipAd is using.

Claude was developed by Anthropic, an AI security company based in San Francisco.
Anthropic was founded in 2021 by Dario Amodei and Daniela Amodei, along with
several former OpenAI employees. The company's goal is to develop AI systems
that are safe, useful, and understandable.
