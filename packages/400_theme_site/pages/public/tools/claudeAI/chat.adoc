[role="dropcap"]
The  is a built-in AI assistant that can answer your questions
about SkipAd directly on the website. It is powered by Claude, an AI LLM
developed by Anthropic. You do not need an account or any setup — just type
a question and get a helpful answer.

// Include sub-documents (if any)

[TIP]
====
The Agent knows all about the SkipAd features, including the video player,
the playlists, how to import and export, series support, and more. It draws
its knowledge from a dedicated knowledge base that is kept up to date
with each release.
====


[role="mt-5"]
== Conversation Thread

All your questions are recorded in the chat history (conversation thread).
The history is stored in the browser, so you don't lose the answers given by
the Agent so far. If you are *not longer interested* in the hisstory, click on
the (red) button *Clear Conversation* to *remove* all entries.
