Bei Video.js werden Features wie der Chapters-Button oder Subtitles-Button
automatisch basierend auf den verfügbaren Daten im Video erkannt und in
der Control Bar angezeigt.

Hier die Details:

## Chapters Button (Kapitel-Navigation)

Der Chapters-Button wird automatisch angezeigt, wenn:

1. **Text Tracks mit `kind="chapters"` vorhanden sind**:
```javascript
// Chapters via <track> Element
<video>
  <track kind="chapters" src="chapters.vtt" srclang="de" label="Kapitel">
</video>

// Oder programmatisch hinzufügen
player.addRemoteTextTrack({
  kind: 'chapters',
  src: 'chapters.vtt',
  srclang: 'de',
  label: 'Kapitel'
});
```

2. **Die WebVTT-Datei korrekt formatiert ist**:
```
WEBVTT

00:00:00.000 --> 00:01:30.000
Einleitung

00:01:30.000 --> 00:03:45.000
Hauptteil

00:03:45.000 --> 00:05:00.000
Fazit
```

## Subtitles Button (Untertitel-Button)

Der Subtitles-Button erscheint automatisch bei:

1. **Text Tracks mit `kind="subtitles"` oder `kind="captions"`**:
```javascript
// Subtitles via HTML
<track kind="subtitles" src="subtitles-de.vtt" srclang="de" label="Deutsch">
<track kind="subtitles" src="subtitles-en.vtt" srclang="en" label="English">

// Programmatisch
player.addRemoteTextTrack({
  kind: 'subtitles',
  src: 'subtitles-de.vtt',
  srclang: 'de',
  label: 'Deutsch'
});
```

## Automatische Erkennung im Detail

Video.js prüft kontinuierlich:

```javascript
// Vereinfachtes Beispiel der internen Logik
player.on('loadedmetadata', function() {
  const textTracks = player.textTracks();
  
  // Prüfung auf Chapters
  const hasChapters = Array.from(textTracks).some(track => 
    track.kind === 'chapters' && track.cues && track.cues.length > 0
  );
  
  // Prüfung auf Subtitles/Captions  
  const hasSubtitles = Array.from(textTracks).some(track => 
    (track.kind === 'subtitles' || track.kind === 'captions') && 
    track.cues && track.cues.length > 0
  );
  
  // Buttons entsprechend ein-/ausblenden
  if (hasChapters) {
    player.getChild('controlBar').getChild('chaptersButton').show();
  }
  
  if (hasSubtitles) {
    player.getChild('controlBar').getChild('subtitlesButton').show();
  }
});
```

## Manuelle Kontrolle

Du kannst das Verhalten auch manuell steuern:

```javascript
// Buttons explizit aktivieren/deaktivieren
const player = videojs('my-video', {
  controls: true,
  // Buttons in Control Bar konfigurieren
  controlBar: {
    chaptersButton: {
      // false = immer verstecken, true = automatische Erkennung
      auto: true
    },
    subtitlesButton: {
      auto: true
    }
  }
});

// Oder nach Initialisierung
player.getChild('controlBar').getChild('chaptersButton').hide();
```

Die automatische Erkennung erfolgt also primär durch das Vorhandensein 
entsprechender Text Tracks mit den korrekten `kind`-Attributen und
verfügbaren Cues.

## Subtitle- oder Caption-Menüs zur Laufzeit in VideoJS hinzufügen

Um bei einem VideoJS-Player zur Laufzeit Untertitel (z.B. aus WEBVTT-Dateien)
hinzuzufügen und die Menüs entsprechend zu aktualisieren, gibt es verschiedene
Ansätze – je nachdem, ob du mit nativen ``-Elementen arbeitest oder eine
eigene Menüführung aufbauen möchtest.

**1. Dynamisches Hinzufügen von Tracks zur Laufzeit**

Du kannst ``-Elemente dynamisch per JavaScript erstellen und dem Video-Tag
hinzufügen. VideoJS erkennt diese und aktualisiert das Untertitel-Menü
automatisch:

```js
var player = videojs('meine-player-id');
var track = document.createElement('track');
track.kind = 'subtitles';
track.label = 'Deutsch';
track.srclang = 'de';
track.src = 'pfad/zur/datei.vtt';
player.el().getElementsByTagName('video')[0].appendChild(track);

// Damit VideoJS das neue Track erkennt:
player.textTracks().addEventListener('addtrack', function() {
  // Optional: Menü aktualisieren oder weitere Aktionen
});
```
Nach dem Hinzufügen erscheint die neue Sprache im Untertitel-Menü des Players[6].

**2. Menü manuell aufbauen (eigene Lösung)**

Falls du ein eigenes Menü unabhängig vom VideoJS-Menü bauen willst, kannst du mit der `textTracks`-API arbeiten:

```js
var video = document.getElementById('meine-video-id');
var captionsMenu = document.createElement('ul');
captionsMenu.className = 'captions-menu';

for (var i = 0; i `-Elementen ist der Standardweg, um VideoJS die Untertitel bekannt zu machen. Das Menü aktualisiert sich automatisch[6].
- Wenn du ein eigenes Menü bauen willst, musst du die `textTracks`-API nutzen und die Anzeige selbst steuern[5].
- Das native VideoJS-Menü kann nicht ohne Weiteres durch eigene Menüeinträge erweitert werden, da es bei bestimmten Aktionen (z.B. beim Laden neuer Quellen) neu aufgebaut wird und eigene Einträge verloren gehen können[6].
- Die Styling- und Einstellungsoptionen für Untertitelmenüs sind begrenzt, wenn du das native Menü verwendest[8].

**Fazit:**  
Am einfachsten ist es, ``-Elemente dynamisch zu erzeugen und dem Video-Tag hinzuzufügen. VideoJS übernimmt dann die Menüführung. Für eigene Menüstrukturen musst du die `textTracks`-API verwenden und das Menü vollständig selbst verwalten[5][6].
