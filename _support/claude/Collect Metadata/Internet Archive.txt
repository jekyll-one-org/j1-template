Es gibt mehrere Wege, Metadaten zu einem Video vom Internet Archive abzurufen:

---

## 1. Metadata API (einfachster Weg)

Die Metadata Read API ist der schnellste und flexibelste Weg, Metadaten für
Items auf archive.org abzurufen.

Das URL-Schema ist simpel:

```
https://archive.org/metadata/{IDENTIFIER}
```

Den **Identifier** findest du in der URL des Videos, z.B.:


`https://archive.org/details/my-video-id` → Identifier = `my-video-id`

Die Antwort kommt als **JSON** und enthält Titel, Beschreibung, Datum,
Dateigröße, Format, etc.


## 2. Nur einen Teil der Metadaten abrufen

Du kannst auch gezielt einzelne Felder abfragen:

```bash
# Nur die item-Metadaten (Titel, Beschreibung, etc.)
curl https://archive.org/metadata/my-video-id/metadata

# Nur die Dateiliste
curl https://archive.org/metadata/my-video-id/files
```

## Hinweis zum Identifier

Falls du nur die Video-URL hast, z.B.:

```
https://archive.org/details/some-cool-video_202301
```
→ der Identifier ist der letzte Teil des Pfades: `some-cool-video_202301`

Brauchst du die Metadaten für einen bestimmten Zweck (z.B. in deinem
J1-Template-Projekt für Video.js)?
